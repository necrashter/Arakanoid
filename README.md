# Arakanoid

Araknoid clone in C++ and SDL 2

## Build on Linux

Install dependencies:
```
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev
```

Build:
```
make
```

Run:
```
cd build
./main
```

If you want to remove all of the compiled objects and executable:
```
make clean
```
